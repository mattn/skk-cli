# skk-cli

![](https://raw.githubusercontent.com/mattn/skk-cli/main/misc/screenshot.png)

## Installation

```
go get github.com/mattn/skk-cli
```

SKK-JISYO.L must be located at `~/.config/skk-cli/SKK-JISYO.L`. For Windows: `%APPDATA%\skk-cli\SKK-JISYO.L`
